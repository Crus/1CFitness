Предназначен для выбора узлов плана обмена (информационных баз).

Вызывается по команде Подобрать или Добавить  из других объектов программы, например, при установке Даты запрета загрузки данных.

В списке выводится:

  • Программа - перечень программ, с которыми настроена синхронизация данных. Информационные базы в каждой программе сгруппированы по типу синхронизации данных - например, без использования правил обмена.
  • Информационная база - перечень информационных баз, с которыми настроена синхронизация данных. Кроме этого, для каждой программы и типа синхронизации предусмотрено значение <Все информационные базы> - с помощью этого значения можно выбрать все информационные базы нужной версии и нужного типа синхронизации.

Выбор узла плана обмена

  • Выделите нужное значение, нажмите Выбрать.

Подбор узлов плана обмена

  • Выделите несколько значений, нажмите Выбрать.
